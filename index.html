前言

前几天在机房突发奇想想要研究研究去年的D1T2（曾经因为写崩而导致只混到了一个省二的一道题）【洛谷P5658】【FAIOJ10191】，于是在早自习的时候手推了一下正确的递推式。

动态规划设计

首先从3条性质入手：

  1. () 是合法括号串；
  2. 如果 A 是合法括号串，则 (A) 是合法括号串。
  3. 如果 A，B 是合法括号串，则 AB 是合法括号串。

首先看第三条性质，对于串()，有1个合法字串，如果在这个串更新为()()，则有3个合法字串，也就是说更新的部分对答案的贡献是2，同理()()()更新的部分对答案的贡献是3。为什么是这样呢？我们不妨用()()更新到()()()这个过程为例，新增加的字串是是以新增的合法字串结尾的所有合法字串。所以我们可以得到这样一个递推式：

g[v]=g[top]+1g[v]=g[top]+1 g[v]=g[top]+1

其中，toptoptop是当前匹配到的合法字串左括号的前驱。这样，我们就得到了以当前位置结尾的所有合法字串的数量。

所以，当前位置的答案为：

f[v]=f[u]+g[v]f[v]=f[u]+g[v] f[v]=f[u]+g[v]

到这里，动态规划的问题就全部结束了，接下来就要思考怎么让DP上树。

实现

在上树之前我们先想一下怎么在链上解决。

括号匹配的方法和暴力的实现方法类似，借助一个栈来解决，在栈中记录左括号的前驱节点的编号。如果当前的节点是右括号，就尝试从栈里弹出一个左括号。

for (int i = 0; i < G[u].size(); i++) {
    int top = -1;
    int v = G[u][i];
    if (s[v] == '(') {
        st.push_back(u);
    } else {
        if (!st.empty()) {
            top = st.back();
            st.pop_back();
            g[v] = g[top] + 1;
        }
    }
    f[v] = f[u] + g[v];
    ans ^= (long long)v * f[v];
    dfs(v);
}

但是，在树上这样就显然是行不通的。因为：

  1. 如果当前节点为右括号，匹配到的左括号从栈里被弹出之后，回溯时需要将弹出的括号放回栈里。
  2. 如果当前节点为左括号，压入栈中之后在回溯是需要弹出。

所以，代码应该改为：

for (int i = 0; i < G[u].size(); i++) {
    int top = -1;
    int v = G[u][i];
    if (s[v] == '(') {
        st.push_back(u);
    } else {
        if (!st.empty()) {
            top = st.back();
            st.pop_back();
            g[v] = g[top] + 1;
        }
    }
    f[v] = f[u] + g[v];
    ans ^= (long long)v * f[v];
    dfs(v);
    if (st.back() == u) //当前节点是左括号，需要弹出当前节点。
        st.pop_back();
    else if (top != -1) //当前节点是右括号，从栈里弹出了一个左括号，需要将其压回栈中。
        st.push_back(top);
}

还有，答案需要开long long，不然会爆掉（算一下数据范围就能知道）。

bool check(int l, int r) {
    stack<bool> st;
    for (int i = l; i <= r; i++) {
        if (s[i] == '(') {
            st.push(1);
        } else {
            if (st.empty())
                return false;
            st.pop();
        }
    }
    return st.empty();
}

ymoi集训-堆和可并堆
2020年08月24日 764 字 大概 3 分钟

堆

一个二叉树结构，可以实现O(log⁡n)O(\log n)O(logn)维护最值。C++ STLpriority_queue实现了大根堆。

push操作

  • 将元素插入数组尾。

  • 从堆尾执行push up操作。

pop操作

  • 将数组首尾元素交换。

  • 删除数组尾元素。

  • 从数组首执行push down操作。

push up操作

  • 直到当前节点已经大于（对于小根堆）/小于（对于大根堆）父节点，或者当前节点为堆顶：

    • 交换当前节点与父节点。

    • 将交换后的父节点视为当前节点。

push down操作

  • 循环执行：

    • 如果当前节点没有子节点，就跳出循环。

    • 取当前节点左右子节点中较大（对于大根堆）/较小（对于小根堆）的那一个，如果大于当前节点，就与之交换，否则跳出循环。

    • 将交换后的子节点视为当前节点。

可并堆

一个左偏树结构，可以O(log⁡n)O(\log n)O(logn)合并的堆。

左偏树

distdistdist的定义：对于一棵二叉树，我们定义外节点为左儿子或右儿子为空的节点，定义一个外节点的distdistdist为111，一个不是外节点的节点distdistdist为其到子树中最近的外节点的距离加一。空节点的distdistdist为000。

左偏树的性质：每一个节点的左儿子distdistdist大于等于右儿子的distdistdist。

因此，左偏树每一个节点的distdistdist都等于其右儿子的distdistdist加一。

合并操作

合并操作的思路就是，递归地合并右儿子与另一个堆。如果不满足左偏性质，就交换两个子节点。具体操作如下：

  • 如果一个堆为空，就返回另一个堆。

  • 取数值较大的一个作为新堆的根。

  • 对右儿子与另一个堆递归执行合并操作。

  • 如果不满足左偏性质，即左儿子的distdistdist大于等于右儿子的distdistdist，就交换左右儿子。

  • 返回新堆的堆顶。

代码：

int merge(int x,int y){
    if(x==0||y==0)return x|y;
    if(f[x].val>f[y].val){
        swap(x,y);
    }
    f[x].rs=merge(f[x].rs,y);
    if(f[f[x].ls].d<f[f[x].rs].d){
        swap(f[x].ls,f[x].rs);
    }
    f[x].d=f[f[x].rs].d+1;
    return x;
}

其他操作

  • push：将单个节点视为一个堆，合并即可。
  • pop：合并根的左右儿子即可。
  • 打标记：与线段树类似，左偏树上也可以通过标记实现任何可以被标记且不改变元素相对大小的操作，如：整堆加/减一个值、乘一个正数。在根打上标记，在执行pop/merge等操作时下传标记即可。

另外，题目中通常需要操作的是「x 所在的堆」，这时一般需要同时维护并查集用于查询「x 所在的堆」的根结点。

强连通分量

定义略。

用Tarjan算法来求解强连通分量，在此之前，先明确这两个定义：

  • 返祖边：由一个点指向其祖先的边。

  • 横插边： 由一个节点指向非祖先节点的边。（只有有向图会出现）

这两种边都是dfs树的非树边。

定义这两个数组：

  • dfndfndfn: dfs序。

  • lowlowlow: 不经横插边所能到达的点中最小的dfndfndfn。

if(!dfn[v]){
    tarjan(v);
    low[u]=min(low[u],low[v]);
}else if(ins[v]){ // 判断不是横插边，即当前子节点v在栈中。
    low[u]=min(low[u],dfn[v]);
}

不难发现，对于某个节点uuu，lowulow_ulowu​表示其深度最小的祖先的dfndfndfn。这样一来，当某个点的dfndfndfn等于lowlowlow时，就说明找到了新的强连通分量。用一个栈来维护强连通分量里的节点，具体操作如下：

  • 更新dfndfndfn和lowlowlow的初值。

  • 当前节点进栈。

  • 遍历当前节点的子节点vvv：

    • 如果vvv未被访问，说明在dfs树上vvv在uuu的子树中，对vvv继续dfs。用lowvlow_vlowv​更新lowulow_ulowu​。

    • 如果vvv被访问且在栈中，说明u→vu\rightarrow vu→v是一条返祖边。用dfnvdfn_vdfnv​更新lowulow_ulowu​。

  • 执行完上述操作后，如果uuu的dfndfndfn和lowlowlow相等，说明找到了一个新的强连通分量，则：

    • 将栈中uuu及其之后的所有节点出栈。

    • 标记强连通分量。

代码：

void tarjan(int u){
    dfn[u]=low[u]=++T;
    ins[u]=true;
    s.push(u);
    for(int i=0;i<G[u].size();i++){
        int v=G[u][i];
        if(!dfn[v]){
            tarjan(v);
            low[u]=min(low[u],low[v]);
        }else if(ins[v]){
            low[u]=min(low[u],dfn[v]);
        }
    }
    if(dfn[u]==low[u]){
        int v;
        cnt++;
        do{
            v=s.top();s.pop();ins[v]=false;
            color[v]=cnt;
        }while(u!=v);
    }
}

由于强连通分量内的点可以互相到达，可以将强连通分量缩成一个点，这样可以得到一个有向无环图。

缩点的流程如下：

  • 遍历所有节点u=1→nu=1\rightarrow nu=1→n：

    • 遍历uuu的所有子节点vvv：

      • 如果uuu和vvv不属于同一个强连通分量，就从uuu所在的强连通分量向vvv所在的强连通分量连边。

割点

在无向图中，对于一个点，如果删除这个点及其连边后，连通块数量增加，那么这就是一个割点。

判断一个点是割点的充分条件如下：

  • 若uuu为根，uuu有两个及以上子节点。

  • 若uuu不为根，在dfs树中uuu的子树中∃v,lowv≥dfnu\exists v, low_v\geq dfn_u∃v,lowv​≥dfnu​。

void tarjan(int u,int fa){ //由于是无向图，需要记录父亲节点
    dfn[u]=low[u]=++T;
    for(int i=0;i<G[u].size();i++){
        int v=G[u][i];
        if(v==fa)continue;
        if(!dfn[v]){
            tarjan(v,u);
            low[u]=min(low[u],low[v]);
            if(low[v]>=dfn[u]&&fa){
                cut[u]=true;
            }
        }else{
            low[u]=min(low[u],dfn[v]);
        }
    }
    if((!fa) && G[u].size()>=2)cut[u]=true;
}

基本概念

一个n×mn \times mn×m的矩阵可以被看做一个nnn行mmm列的数阵,一般用一个大写字母表示：

A=[114514],B=[10982167]A=\begin{bmatrix}1&1&4\\ 5&1&4\end{bmatrix}, B=\begin{bmatrix}1&0\\ 9&8\\ 2&1\\ 6&7\end{bmatrix} A=[15​11​44​],B=⎣⎢⎢⎡​1926​0817​⎦⎥⎥⎤​

矩阵之间可以进行加减乘除运算，下面研究矩阵的乘法运算。

对于两个矩阵AAA和BBB，只有AAA的行数等于BBB的列数时，才能进行乘法。设A的大小n×mn \times mn×m，BBB的大小为m×pm \times pm×p为乘法的定义如下：

Ci,j=∑k=1mAi,k⋅Bk,j(1≤i≤n,1≤j≤p)C_{i,j}=\sum_{k=1}^mA_{i,k}\cdot B_{k,j}(1\leq i \leq n,1\leq j \leq p) Ci,j​=k=1∑m​Ai,k​⋅Bk,j​(1≤i≤n,1≤j≤p)

时间复杂度为O(n3)O(n^3)O(n3)（假设nnn, mmm, ppp同阶）。

矩阵乘法满足结合律，因此我们可以……

矩阵快速幂

和实数的快速幂求法完全一样：

int ksm(int a,int b){

树链剖分解决什么问题

先看这样的一个问题：

给出一棵nnn个结点, 每个结点有点权, 要求支持:

1 u v x 将树上u→vu\rightarrow vu→v这条链的所有结点点权增加xxx
2 u x 将uuu的子树的所有结点的点权增加xxx
3 u v 询问u→vu\rightarrow vu→v这条链的所有结点点权和
4 u 询问uuu的子树的所有结点的点权和

n,q≤1×105n,q\leq 1\times 10^5n,q≤1×105

这个问题看起来很像线段树的题，不过这是在树上操作，所以我们要用树链剖分。

轻重链剖分

简称HLD(Heavy Light Decomposition)。

基本概念

先做出如下定义：

  • siz[i]siz[i]siz[i]表示iii结点的子树大小
  • dep[i]dep[i]dep[i]表示iii结点的深度(定义根的深度为1)
  • fa[i]fa[i]fa[i]表示iii结点的父结点编号

对于所有v∈{v∣u→v}v\in \{v|u\rightarrow v\}v∈{v∣u→v}，称siz[v]siz[v]siz[v]最大的为uuu的重儿子，边u→vu\rightarrow vu→v为重边。

除重边以外的边称为轻边。

再做出如下定义：

  • son[i]son[i]son[i]表示iii结点的重儿子编号
  • top[i]top[i]top[i]表示iii结点所在重链顶端结点的编号
  • id[i]id[i]id[i]表示iii结点按照重儿子优先顺序的dfs序号

代码：

int lca(int u,int v){
    while(top[u]!=top[v]){
        if(dep[top[u]]<dep[top[v]])swap(u,v);
        u=fa[top[u]];
    }
    return dep[u]<dep[v]?u:v;
}

用求LCA的方法，就可以进行线段树的查询和更新。在向上跳之前更新即可。这时所更新的链为top[u]→utop[u]\rightarrow utop[u]→u，这条链在线段树上所对应的区间为[id[top[u]],id[u]][id[top[u]],id[u]][id[top[u]],id[u]]。

当然，在最后不要忘记对u→vu\rightarrow vu→v这条链进行操作。

代码：

int query_path(int u,int v){
    int ans=0;
    while(top[u]!=top[v]){
        if(dep[top[u]]<dep[top[v]])swap(u,v);
        ans+=query(1,1,n,id[top[u]],id[u]);
        u=fa[top[u]];
    }
    if(dep[u]>dep[v])swap(u,v);
    ans+=query(1,1,n,id[u],id[v]);
    return ans;
}

void update_path(int u,int v,int val){
    while(top[u]!=top[v]){
        if(dep[top[u]]<dep[top[v]])swap(u,v);
        update(1,1,n,id[top[u]],id[u],val);
        u=fa[top[u]];
    }
    if(dep[u]>dep[v])swap(u,v);
    update(1,1,n,id[u],id[v],val);
}

以上就是轻重链剖分的基本操作。
