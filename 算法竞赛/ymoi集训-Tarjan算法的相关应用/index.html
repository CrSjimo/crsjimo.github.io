强连通分量

定义略。

用Tarjan算法来求解强连通分量，在此之前，先明确这两个定义：

  • 返祖边：由一个点指向其祖先的边。

  • 横插边： 由一个节点指向非祖先节点的边。（只有有向图会出现）

这两种边都是dfs树的非树边。

定义这两个数组：

  • dfndfndfn: dfs序。

  • lowlowlow: 不经横插边所能到达的点中最小的dfndfndfn。

if(!dfn[v]){
    tarjan(v);
    low[u]=min(low[u],low[v]);
}else if(ins[v]){ // 判断不是横插边，即当前子节点v在栈中。
    low[u]=min(low[u],dfn[v]);
}

不难发现，对于某个节点uuu，lowulow_ulowu​表示其深度最小的祖先的dfndfndfn。这样一来，当某个点的dfndfndfn等于lowlowlow时，就说明找到了新的强连通分量。用一个栈来维护强连通分量里的节点，具体操作如下：

  • 更新dfndfndfn和lowlowlow的初值。

  • 当前节点进栈。

  • 遍历当前节点的子节点vvv：

    • 如果vvv未被访问，说明在dfs树上vvv在uuu的子树中，对vvv继续dfs。用lowvlow_vlowv​更新lowulow_ulowu​。

    • 如果vvv被访问且在栈中，说明u→vu\rightarrow vu→v是一条返祖边。用dfnvdfn_vdfnv​更新lowulow_ulowu​。

  • 执行完上述操作后，如果uuu的dfndfndfn和lowlowlow相等，说明找到了一个新的强连通分量，则：

    • 将栈中uuu及其之后的所有节点出栈。

    • 标记强连通分量。

代码：

void tarjan(int u){
    dfn[u]=low[u]=++T;
    ins[u]=true;
    s.push(u);
    for(int i=0;i<G[u].size();i++){
        int v=G[u][i];
        if(!dfn[v]){
            tarjan(v);
            low[u]=min(low[u],low[v]);
        }else if(ins[v]){
            low[u]=min(low[u],dfn[v]);
        }
    }
    if(dfn[u]==low[u]){
        int v;
        cnt++;
        do{
            v=s.top();s.pop();ins[v]=false;
            color[v]=cnt;
        }while(u!=v);
    }
}

缩点

由于强连通分量内的点可以互相到达，可以将强连通分量缩成一个点，这样可以得到一个有向无环图。

缩点的流程如下：

  • 遍历所有节点u=1→nu=1\rightarrow nu=1→n：

    • 遍历uuu的所有子节点vvv：

      • 如果uuu和vvv不属于同一个强连通分量，就从uuu所在的强连通分量向vvv所在的强连通分量连边。

代码：

void suodian(){
    for(int u=1;u<=n;u++){
        c_siz[color[u]]+=w[u];
        for(int i=0;i<G[u].size();i++){
            int v=G[u][i];
            if(color[u]==color[v])continue;
            Gs[color[u]].push_back(color[v]);
        }
    }
}

割点

在无向图中，对于一个点，如果删除这个点及其连边后，连通块数量增加，那么这就是一个割点。

判断一个点是割点的充分条件如下：

  • 若uuu为根，uuu有两个及以上子节点。

  • 若uuu不为根，在dfs树中uuu的子树中∃v,lowv≥dfnu\exists v, low_v\geq dfn_u∃v,lowv​≥dfnu​。

代码：

void tarjan(int u,int fa){ //由于是无向图，需要记录父亲节点
    dfn[u]=low[u]=++T;
    for(int i=0;i<G[u].size();i++){
        int v=G[u][i];
        if(v==fa)continue;
        if(!dfn[v]){
            tarjan(v,u);
            low[u]=min(low[u],low[v]);
            if(low[v]>=dfn[u]&&fa){
                cut[u]=true;
            }
        }else{
            low[u]=min(low[u],dfn[v]);
        }
    }
    if((!fa) && G[u].size()>=2)cut[u]=true;
}
