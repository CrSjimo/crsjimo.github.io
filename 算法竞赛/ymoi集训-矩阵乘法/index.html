基本概念

一个n×mn \times mn×m的矩阵可以被看做一个nnn行mmm列的数阵,一般用一个大写字母表示：

A=[114514],B=[10982167]A=\begin{bmatrix}1&1&4\\ 5&1&4\end{bmatrix}, B=\begin{bmatrix}1&0\\ 9&8\\ 2&1\\ 6&7\end{bmatrix} A=[15​11​44​],B=⎣⎢⎢⎡​1926​0817​⎦⎥⎥⎤​

矩阵之间可以进行加减乘除运算，下面研究矩阵的乘法运算。

对于两个矩阵AAA和BBB，只有AAA的行数等于BBB的列数时，才能进行乘法。设A的大小n×mn \times mn×m，BBB的大小为m×pm \times pm×p为乘法的定义如下：

Ci,j=∑k=1mAi,k⋅Bk,j(1≤i≤n,1≤j≤p)C_{i,j}=\sum_{k=1}^mA_{i,k}\cdot B_{k,j}(1\leq i \leq n,1\leq j \leq p) Ci,j​=k=1∑m​Ai,k​⋅Bk,j​(1≤i≤n,1≤j≤p)

时间复杂度为O(n3)O(n^3)O(n3)（假设nnn, mmm, ppp同阶）。

矩阵乘法满足结合律，因此我们可以……

矩阵快速幂

和实数的快速幂求法完全一样：

int ksm(int a,int b){
    int ans = 1;
    for(;b;a=(a*a)%MOD,b>>=1){
        if(b&1)ans=(ans*a)%MOD;
    }
    return ans;
}

只不过把初值赋成单位矩阵即可。

若存在e∈Se\in Se∈S使∀x∈S\forall x\in S∀x∈S与eee进行某种运算后得到这个元素本身，则eee被称为单位元。

在R\RR中, 111即为乘法法则下的单位元。

单位矩阵是主对角线为111，其余均为000的矩阵：

I=[10⋯001⋯0⋮⋮⋱⋮00⋯1]I=\begin{bmatrix}1&0&\cdots&0\\ 0&1&\cdots&0\\ \vdots&\vdots&\ddots&\vdots\\ 0&0&\cdots&1\end{bmatrix} I=⎣⎢⎢⎢⎡​10⋮0​01⋮0​⋯⋯⋱⋯​00⋮1​⎦⎥⎥⎥⎤​

所以把初值赋为单位矩阵即可。

1

题目

斐波那契数列:f1=f2=1,fn=fn−1+fn−2(n>2)f_1=f_2=1,f_n=f_{n-1}+f_{n-2}(n>2)f1​=f2​=1,fn​=fn−1​+fn−2​(n>2)

求 fnmod  109+7f_n \mod 10^9+7fn​mod109+7

n≤1018n\leq10^{18}n≤1018

解法

易证明：

[fn−2fn−1]⋅[0111]=[fn−1fn]\begin{bmatrix}f_{n-2}&f_{n-1}\end{bmatrix}\cdot\begin{bmatrix}0&1\\ 1&1\end{bmatrix}=\begin{bmatrix}f_{n-1}&f_{n}\end{bmatrix} [fn−2​​fn−1​​]⋅[01​11​]=[fn−1​​fn​​]

其中，将[0111]\begin{bmatrix}0&1\\ 1&1\end{bmatrix}[01​11​]称为转移矩阵，记作BBB。

则有：

[fn−1fn]=[f1f2]⋅Bn−2\begin{bmatrix}f_{n-1}&f_{n}\end{bmatrix}=\begin{bmatrix}f_{1}&f_{2}\end{bmatrix}\cdot B^{n-2} [fn−1​​fn​​]=[f1​​f2​​]⋅Bn−2

就是这么神奇。

其他的问题解法大致相同，推导转移矩阵就行。（但是要怎么推导呢？）

邻接矩阵Gu,vG_{u,v}Gu,v​表示从uuu到vvv经过一条边的路径数，根据矩阵乘法的定义不难看出：Gu,vkG^k_{u,v}Gu,vk​表示从uuu到vvv经过kkk条边的路径数，

4

题目

给定一张TTT条边的无向连通图，求从uuu到vvv经过kkk条边的最短路长度。

解法

定义一个矩阵新运算：

Ci,j=min⁡k=1→mAi,k⋅Bk,j(1≤i≤n,1≤j≤p)C_{i,j}=\min_{k=1\rightarrow m}A_{i,k}\cdot B_{k,j}(1\leq i \leq n,1\leq j \leq p) Ci,j​=k=1→mmin​Ai,k​⋅Bk,j​(1≤i≤n,1≤j≤p)

这个运算类似于矩阵乘法，且满足结合律，可以使用矩阵快速幂。求法与例题3类似。
