# 堆

一个二叉树结构，可以实现$O(\log n)$维护最值。C++ STL`priority_queue`实现了大根堆。

## push操作

- 将元素插入数组尾。

- 从堆尾执行push up操作。

## pop操作

- 将数组首尾元素交换。

- 删除数组尾元素。

- 从数组首执行push down操作。

## push up操作

- 直到当前节点已经大于（对于小根堆）/小于（对于大根堆）父节点，或者当前节点为堆顶：

    - 交换当前节点与父节点。

    - 将交换后的父节点视为当前节点。

## push down操作

- 循环执行：

    - 如果当前节点没有子节点，就跳出循环。

    - 取当前节点左右子节点中较大（对于大根堆）/较小（对于小根堆）的那一个，如果大于当前节点，就与之交换，否则跳出循环。

    - 将交换后的子节点视为当前节点。

# 可并堆

一个左偏树结构，可以$O(\log n)$合并的堆。

## 左偏树

$dist$的定义：对于一棵二叉树，我们定义**外节点**为左儿子或右儿子为空的节点，定义一个外节点的$dist$为$1$，一个不是外节点的节点$dist$为其到子树中最近的外节点的距离加一。空节点的$dist$为$0$。

左偏树的性质：每一个节点的左儿子$dist$大于等于右儿子的$dist$。

因此，**左偏树每一个节点的$dist$都等于其右儿子的$dist$加一。**

## 合并操作

合并操作的思路就是，递归地合并右儿子与另一个堆。如果不满足左偏性质，就交换两个子节点。具体操作如下：

- 如果一个堆为空，就返回另一个堆。

- 取数值较大的一个作为新堆的根。

- 对右儿子与另一个堆递归执行合并操作。

- 如果不满足左偏性质，即左儿子的$dist$大于等于右儿子的$dist$，就交换左右儿子。

- 返回新堆的堆顶。

代码：

```C++
int merge(int x,int y){
    if(x==0||y==0)return x|y;
    if(f[x].val>f[y].val){
        swap(x,y);
    }
    f[x].rs=merge(f[x].rs,y);
    if(f[f[x].ls].d<f[f[x].rs].d){
        swap(f[x].ls,f[x].rs);
    }
    f[x].d=f[f[x].rs].d+1;
    return x;
}
```

## 其他操作

- `push`：将单个节点视为一个堆，合并即可。
- `pop`：合并根的左右儿子即可。
- 打标记：与**线段树**类似，左偏树上也可以通过标记实现任何可以被标记且**不改变元素相对大小的操作**，如：整堆加/减一个值、乘一个正数。在根打上标记，在执行`pop`/`merge`等操作时下传标记即可。

另外，题目中通常需要操作的是「x 所在的堆」，这时一般需要同时维护**并查集**用于查询「x 所在的堆」的根结点。
